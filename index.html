Printer Problems

General solution to printer problems, June 16, 2019

-Check for any blinking lights

Projector Problems

General solution to projector problems, June 16, 2019

-Check to see if bult replacement light is on
-Make sure projector VGA cable (black cable with blue connector) is properly plugged into computer
-Try holding down the power button and help button on the remote as you try to power up the projector
